# "Nested Cubes" example for Retro Debugger WebSocket API client

# Usage:

To run example, execute:
- Using Makefile: `make run`
- Using Go: `go run .`

To build executable binary:
- Using Makefile: `make build`
- Using Go: `go build -o bin/cubes .`
